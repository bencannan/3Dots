# 3Dots – From idea to MVP in 15 days

Minimal, dark-theme marketing site for **3Dots**, a tech studio that helps founders go from idea to MVP in 15 days (post-design approval). The site uses plain HTML, CSS, and a small amount of JavaScript for animations and micro-interactions.

## Structure

- `index.html` – Single-page layout with all sections:
  - Hero with AI-inspired animated visual and CTA
  - Problem, Approach
  - 15-day MVP Framework (timeline layout)
  - Why this matters for founders
  - Core capabilities (cards)
  - Who we work with
  - Final CTA
- `styles.css` – Dark UI, typography, layout, hover states, scroll and orbit animations.
- `script.js` – Smooth scrolling, scroll-reveal effects, and hero code-typing animation.

## Running locally

You can simply open `index.html` directly in your browser (double-click or drag into a tab).

If you prefer a tiny static server (recommended for future extensions):

```bash
cd "3 Dots Website"
npm install
npm run start
```

Then open the printed `http://localhost:PORT` in your browser.
